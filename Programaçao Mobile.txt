Dart é uma
linguagem de programação otimizada para o desenvolvimento rápido de aplicativos (UI) em múltiplas plataformas (móvel, web, desktop), criada pelo Google em 2011 e baseada em orientação a objetos.

O Flutter
é um framework de desenvolvimento de interface de usuário (UI) de código aberto criado pelo Google, utilizado para construir aplicativos compilados nativamente para mobile (Android, iOS), web e desktop a partir de uma única base de código.

Você vai precisar:

Visual Studio Code ou Android Studio

Para iOS:
Xcode (só roda em Mac)

⚠️ Importante:
👉 Para publicar em iOS, você precisa de um Mac (ou usar serviços na nuvem)

Se o app tiver login, banco de dados, etc:

Opções:
Firebase (mais fácil)
Supabase
Backend próprio (Node.js, Python, etc.)
